# 🛡️ Firewall

> A firewall protects a network or system by monitoring and controlling incoming and outgoing network traffic based on predefined security rules.

---

## 📖 What is a Firewall?

A **Firewall** is a security device or software that filters network traffic and decides whether to **allow** or **block** connections based on configured rules.

---

## 🎯 Why is a Firewall Important?

- 🚫 Blocks unauthorized access
- 🔒 Protects systems from attacks
- 🌐 Monitors network traffic
- 🛡️ Enforces security policies

---

## ⚙️ How a Firewall Works

1. A packet reaches the firewall.
2. The firewall checks its security rules.
3. The packet is either:
   - ✅ Allowed
   - ❌ Blocked

---

## 📚 Types of Firewalls

### 🔹 Packet Filtering Firewall
Filters packets based on IP addresses, ports, and protocols.

### 🔹 Stateful Firewall
Tracks active connections and allows only legitimate traffic.

### 🔹 Application Firewall
Inspects application-level traffic such as HTTP and HTTPS.

---

## ✅ Advantages

- Protects against unauthorized access
- Filters malicious traffic
- Improves network security
- Helps prevent attacks

---

## ⚠️ Limitations

- Cannot stop every cyberattack
- Incorrect rules can block legitimate traffic
- Requires proper configuration
