# Networking-fundamentals
A collection of networking concepts and hands-on labs as part of my cybersecurity journey.

## 📚 Topics

| Topic | Description |
|-------|-------------|
| 🌐 **[TCP/IP](01-TCP-IP-Model/)** | Learn the TCP/IP model, layers, and protocol suite. |
| 📚 **[OSI Model](02-OSI-model/)** | Understand the seven-layer OSI reference model. |
| 🌍 **[DNS](03-DNS/)** | Learn how domain names are translated into IP addresses. |
| 🔐 **[SSH](04-SSH/)** | Secure remote access and encrypted communication. |
| 📡 **[ARP](05-ARP/)** | Discover how IP addresses are mapped to MAC addresses. |
| 🌍 **[NAT](06-NAT/)** | Explore Network Address Translation and IP conservation. |
| 🛡️ **[Firewall](07-Firewall/)** | Learn how firewalls filter and protect network traffic. |
| 📡 **[DHCP](08-DHCP/)** | Understand automatic IP address assignment using DHCP. |
