# DNS
**DNS** stands for Domain Name System.  
- It translates easy to read domain-names(like google.com) into computer friendly IP address.
- This allows user to browse web without the need to memorize IP address.

### Records in DNS
|Record|Description|
|-|-|
|A| IPv4 record|
|AAAA|IPv6 record|
|CNAME (Canonical Name)|Forwards domain and subdomain to another domain|
|MX (Mail Exchanger)|Directs mail to email server|
|TXT (Text)|Any text by admin|
|NS (Name Server)|Name server by DNS entry|
|SOA (Start Of Authority)|Admin information about a domain||
|SRV (Service location)|Specify port for specific service|
|PTR (Pointer)|Provides domain name in reverse-lookups|
